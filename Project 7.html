Project Overview

Background

OCA is a Communication Platform-as-a-Service platform that enables businesses to send messages through WhatsApp, SMS, Email, and Call. Previously, each channel was monitored separately, making it difficult for stakeholders to evaluate cross-channel performance, delivery effectiveness, revenue contribution, and user behavior in one integrated view. This project aimed to create a unified performance dashboard to support faster and more accurate business decision-making.

Method

  • Conducted data understanding and preparation to ensure channel data could be analyzed consistently.
  • Performed exploratory analysis to identify cross-channel performance patterns, user behavior, and operational risks.
  • Created a unified dashboard to monitor WhatsApp, SMS, Email, and Call performance in one integrated view.
  • Standardized key metrics across channels, including revenue, transaction volume, message volume, success rate, failed rate, pending rate, active users, average messages per user, average active days, and billable rate.
  • Generated insights and recommendations to support faster, more accurate, and actionable decision-making.

Result

  • Built a dashboard showing that overall channel performance remained relatively healthy, with a 78.65% billable rate, 79.54% success rate, and only 1.29% failed rate.
  • Found that revenue was highly concentrated in WhatsApp and SMS, which together contributed around 86% of total revenue.
  • Identified SMS as the most efficient billable channel, with a 99.67% billable rate.
  • Detected Call as the weakest-performing channel due to a high 77.55% pending rate.
  • Uncovered user concentration risk, where user_3, user_2, and user_1 dominated both revenue and transaction volume.
  • Recommended prioritizing top-user retention, protecting WhatsApp and SMS as core revenue channels, investigating Call pending issues, and evaluating Email monetization.
